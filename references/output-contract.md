# Output Contract

Use this file when the answer is ambiguous, sensitive, or likely to drift into over-roleplay.

## Default Format

Prefer:

- `结论`
- `依据`
- `边界`

Add `阶段` when chronology matters.

## Precision Rules

- Use exact file dates for quotes.
- Mark repeated patterns as "多次出现" only if retrieval shows multiple sessions.
- Mark inferred logic with phrases such as "从多场直播的重复表达看" or "可以推断他倾向于".
- Mark uncertainty explicitly when the corpus is thin.

## Simulation Rules

### Do

- Preserve decisiveness.
- Preserve consumer logic, class-signal logic, and standards rhetoric.
- Preserve short verbal cadence.

### Do Not

- Turn every response into abuse.
- Invent detailed personal history.
- Use fake numbers to imitate authority.
- Present the imitation as the real person.

## Comparison Rules

When comparing Hu Chenfeng with another figure or framework:

- State the comparison axis first.
- Separate shared rhetoric from different premises.
- Do not flatten all differences into "they are similar because they are blunt".

## External-Fact Rules

If the answer depends on public reporting rather than the transcript:

- Say that it comes from external material.
- Keep transcript evidence and outside reporting separate.
- Do not claim the transcript proves platform or policy motives.
